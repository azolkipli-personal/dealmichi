🤝 DealMichi Privacy Policy

Last updated: June 18, 2026

Overview

DealMichi is a sales pipeline tracking application that runs entirely on your device. We do not collect, store, transmit, or share any personal data. This policy exists to be transparent about what the app does with your information.

Data Storage

All data you enter into DealMichi (deals, contacts, notes, settings) is stored exclusively on your device using:

  • IndexedDB (via Dexie.js) — your deal database
  • localStorage — app preferences and stage configuration

This data never leaves your device. There are no servers, no cloud sync, no accounts, and no backend.

Internet Access

DealMichi does not make any network requests to external servers for its core functionality. The only network access occurs when:

  • You manually export/import a JSON backup file (client-side file operations only)
  • You load the app icon from the manifest (same origin)

Third-Party Services

DealMichi uses the following open-source libraries, which run entirely client-side and do not send data anywhere:

  • Dexie.js — IndexedDB wrapper (dexie.org)
  • Marked — Markdown parser (marked.js.org)

Both libraries are loaded from CDNs during app initialization but do not transmit your data.

Data Export & Backup

You can export all your data as a JSON file at any time using the Export button. This is a client-side file download — no data is sent to any server.

Children's Privacy

DealMichi is not directed at children under 13. We do not knowingly collect any information from children.

Changes to This Policy

If this policy changes, the "Last updated" date at the top will be revised. Since the app has no network connectivity to check for updates, please review this page periodically.
