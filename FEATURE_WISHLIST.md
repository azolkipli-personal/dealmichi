# BerKanban PWA — Feature Wishlist

> Refer to this for future features. Prioritised by value vs effort.

## 🔥 High Value, Low Effort

- [x] Dashboard stats bar (Total Pipeline, Win Rate, Avg Deal, Active)
- [x] Undo delete (toast with undo button, 5s timeout)
- [x] Mobile drag & drop (long-press → drag between columns)
- [ ] Auto theme follow system (respect phone's dark/light setting by default)

## 📈 Medium Value

- [ ] Reorder cards within columns (pin important deals to top)
- [ ] Activity log (auto-track when deals moved, edited, etc.)
- [ ] Markdown preview (see formatted notes in edit modal)
- [ ] Filter by tags (custom labels like "Urgent", "Q3", "New")
- [ ] Notes export per deal (export single deal as .md file)

## 🚀 Stretch Goals

- [ ] Cloud sync via Google Drive (sync deals between phone + tablet)
- [ ] Android widget (pipeline summary on home screen)
- [ ] Share deal as text (Android share sheet → WhatsApp/email)
- [ ] Swipe to move (swipe left → Closing, right → Won)
